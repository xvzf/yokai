    Applications demos

    Yokai provides a showroom for its demo applications.

    HTTP application demo

    This demo application is a simple HTTP REST API (CRUD) to manage gophers.

    It provides:

    • a Yokai application container, with
      • the fxhttpserver module to expose the REST API
      • the fxorm module to enable database interactions
    • a MySQL container to store the gophers

    Available on GitHub.

    Worker application demo

    This demo application provides is a simple worker example subscribing to Pub/Sub.

    It provides:

    • a Yokai application container, with:
      • the fxhttpserver module to expose a Pub/Sub publication endpoint
      • the fxworker module to provide a worker running a Pub/Sub subscriber
    • a Pub/Sub emulator container

    Available on GitHub.
